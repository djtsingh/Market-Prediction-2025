Project: Hull Tactical - Market Prediction

This repository contains code and experiments for the Hull Tactical Kaggle competition. The goal is to predict daily excess returns and produce an allocation between 0 and 2 while controlling volatility.

Structure (starter):
- data/                # place raw train/test here (already present)
- src/                 # python package
- notebooks/           # EDA and experiments
- experiments/         # experiment outputs and logs
- configs/             # configuration files
- scripts/             # helper scripts (env setup, run)
